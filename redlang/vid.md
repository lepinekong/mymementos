### Set title, size, and background of Window

![https://i.imgur.com/a26B3xe.png](https://i.imgur.com/a26B3xe.png)
                    


```

title: "hello world"
win-size: 640x480
background-color: black

win: compose [
    title (title)
    size (win-size)
    backdrop (background-color)
]
view layout win
        
```
